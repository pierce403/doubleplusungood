# Notes index

Brief guide to the research artifacts in this directory.

- `full-timeline.md` — Complete chronological git log of the vendored `notepad-plus-plus/` repo (6,600+ commits), oldest first.
- `auto-update-timeline.md` — Evolution of the updater (WinGUp/gup.exe): introduction (2007), UX controls, domain pinning (8.8.8), signature enforcement (8.8.9), and 2025 malware-incident IOCs/triage tips.
- `cert-and-ssl-timeline.md` — History of update URLs, HTTPS adoption, signing practices, certificate verification, and the 8.8.9 hardening; confirms no root-CA installation by installers.
- `news-coverage-malware-updater.md` — Press roundup (Register, Cybernews, Yahoo/Tech, Heise) with cited sources, IOCs, and how coverage maps to repo commits.
- `incident-response-playbook-notepadpp-updater.md` — Step-by-step IR playbook for the updater compromise: identification, containment, eradication, recovery, and hardening guidance.
- `evilgrade-2008-notes.md` — Summary of the 2008 Evilgrade/CVE-2008-3436 updater spoofing (pre-4.8.1) and how it relates to later hardening.
